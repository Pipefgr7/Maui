# Taller .NET MAUI — Programación Móvil

**Universidad Libre Seccional Pereira**  
Facultad de Ingeniería · Programación Móvil

## 📖 Descripción

Repositorio con el desarrollo de 5 aplicaciones móviles realizadas en **.NET MAUI** como parte de un taller guiado de Programación Móvil.  
El objetivo es practicar desde interfaces básicas hasta persistencia de datos y arquitectura MVVM.  

### Conceptos trabajados

- Interfaces gráficas con XAML  
- Eventos y controles  
- Data Binding  
- Patrón MVVM  
- CollectionView  
- Navegación con Shell  
- Persistencia local con SQLite  
- CRUD completo  
- Manejo de errores  

## 🛠️ Tecnologías utilizadas

- .NET MAUI  
- C#  
- XAML  
- SQLite  
- MVVM  
- CommunityToolkit.Mvvm  
- Visual Studio 2022 / 2026  
- Android Emulator / MEmu  

## 📂 Proyectos

### 1️⃣ MiPrimeraAppMAUI

Aplicación básica de bienvenida para comprender los fundamentos de .NET MAUI.

**Funcionalidades:**

- Ingreso de nombre  
- Saludo personalizado  
- Validación de campos  

**Controles usados:** `Label`, `Entry`, `Button`, eventos `Clicked`.

---

### 2️⃣ MiniCalculadoraMAUI

Calculadora básica con distintos controles visuales.

**Funcionalidades:**

- Suma, resta, multiplicación y división  
- Validaciones de entrada  
- Manejo de división entre cero  

**Tecnologías:** `Grid` layout, `Picker`, `Entry`, eventos de botones.

---

### 3️⃣ ClimaAppMAUI

Aplicación de clima simulada con Data Binding y arquitectura MVVM.

**Funcionalidades:**

- Actualización dinámica del clima  
- Cambio de temperatura, humedad y condiciones climáticas  

**Tecnologías:** MVVM, `BindingContext`, `INotifyPropertyChanged`, Data Binding.

---

### 4️⃣ ListaTareasMAUI

Gestión de tareas pendientes con manejo de colecciones.

**Funcionalidades:**

- Agregar y eliminar tareas  
- Marcar tareas como completadas  
- Actualización automática de la UI  

**Tecnologías:** `ObservableCollection`, `CollectionView`, CommunityToolkit.Mvvm, `ICommand`.

---

### 5️⃣ AgendaSQLiteMAUI

Proyecto integrador con persistencia de datos usando SQLite.

**Funcionalidades:**

- Crear, editar y eliminar contactos  
- Buscar contactos  
- Persistencia local  
- Gestos con `SwipeView`  

**Tecnologías:** `SQLiteAsyncConnection`, MVVM, CRUD, `SearchBar`, `SwipeView`.

## ▶️ Cómo ejecutar

1. **Clonar el repositorio**

   ```bash
   git clone https://github.com/tu-usuario/tu-repo-maui.git
   ```

2. **Abrir el proyecto en Visual Studio 2022/2026**

   - Abre el archivo `.sln` del proyecto que desees ejecutar.

3. **Seleccionar plataforma y ejecutar**

   - Elige entre **Windows Machine**, **Android Emulator** o **MEmu** y presiona `F5`.
